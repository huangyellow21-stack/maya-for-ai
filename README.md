# AIFORMAYA - Maya AI 智能助手

AIFORMAYA 是一款为 Autodesk Maya (2020+) 定制的全功能、高扩展性 AI 智能助手插件。它能够理解自然语言指令，在 Maya 内部自动执行繁琐的操作，甚至提供专业的生产指导。系统采用大语言模型（如 DeepSeek 和 Gemini）作为核心引擎，就像拥有一个专属的资深技术总监（TD）随时待命。

---

## ✨ 核心特性

### 1. 极致灵活的动态执行流 🚀
摆脱传统“白名单 API”的死板限制。当遇到内置工具链无法直接完成的复杂指令（例如：“建10个小球摆成一排”、“把所有灯光颜色改红”、“给所有动画曲线做平滑”）时，AI 会在后台自动编写高质量的 Maya Python (`cmds`, `mel`) 脚本并即时运行，满足几乎任何天马行空的创意需求。

### 2. 现代优雅的交互体验 (Modern UI) 🎨
- **精美结构化卡片**：彻底告别传统 AI 助手的枯燥纯文本。操作回执、危险警告、系统状态全部采用现代化的卡片式 UI 渲染（✅/❌ 图标提示），一目了然。
- **纯净的聊天体验**：系统会自动在后台过滤 DeepSeek 等推理模型的 `<think>` 长篇推理过程，只为你呈现最终结论和操作结果，保持面板极致清爽。
- **无感多轮调度**：面对繁杂的长链条步骤，底层的重试与死循环拦截机制让 AI 思考得平滑且快速。
- **自动持久化**：关闭面板或重启 Maya 后，交互记录无损恢复。
- **双工作模式切换**：支持随时在“查询模式”与“自动化编辑执行模式”之间丝滑切换，互不干涉。

### 3. 生产安全第一 (Safety First) 🛡️
系统具备极强的主动安全意识。当解析到诸如“删除整个场景”、“重置进度”或高风险的破坏性操作时，AI 将暂缓执行危险命令，并在 UI 面板上弹出**防灾二次确认卡片**（Confirmation Card），让你从容决定是否放行。

### 4. 先进的上下文记忆 (Entity Memory) 🧠
打破了大模型无法连续跨句子选中物体的魔咒。内置的 `EntityMemory` 会记录每次代偿操作产生的新物体（例如：“创建一个摄像机” -> “让**它**抖动”）。系统能在多轮对话中隐式映射上下文，大幅提升指令对话的连贯性。

### 5. 高内聚的本机 AI 网关 🔌
- **一键无缝启动**：告别繁琐的终端命令，直接在 Maya 侧边栏面板一键启动本地 Python 3 网关进程。
- **智能排障提示**：当网关意外断开或端口被占用时，UI 会展示明显的断连错误卡片，并附带针对性的直接排障方案。

### 6. 实用的内置生产工具 (Robust Tools) 🛠️
除了动态写代码，助手底层封装了一批极致契合生产直觉的**高频“一键”动作**，全部支持**基于当前选中项操作**、**结构化成功回执**，并且统统可以 **Ctrl+Z 瞬间撤销**，让你在生产中用得大胆无畏：
- **场景优化 (`cleanup_scene`)**：一键清除无用历史、冻结坐标、居中枢轴。未选物体时还会防弹窗告警全量清理。
- **智能打组 (`group_and_center`)**：极速对零散模型打组，并自动把枢轴放置到模型簇的绝对正中心。
- **随机散布 (`randomize_transforms`)**：精准对一批选中物体（如石头、树木）赋予位置、旋转和缩放的随机扰动。
- **神级涂装 (`assign_color_materials`)**：一句话瞬间为百个物体分配单独的纯色材质球或随机涂色，极其高效。
- **摄影棚布光与运镜 (`three_point_lighting` / `turntable`)**：喊一句“搭一个三点布光”或“环绕展示它”，一秒出效果。

## 🤯 为什么它不是脚本？(AI vs Script)

许多人可能会问：“这不就是把 Maya 脚本绑在对话框上吗？”
**绝不是！** AIFORMAYA 拥有传统脚本无法企及的“感知力”与“决策力”：

| 对比维度 | 传统 Maya 脚本 / 宏 | AIFORMAYA 智能助手 |
| :--- | :--- | :--- |
| **参数化设定** | 只能写死参数（如灯光强度固定、距离定死为 20） | **智能场景自适应**：自动读取你所选物体的包围盒尺寸（BBox），从而动态决定相机该拉多远、灯光该打多亮。它在“看”你的场景。 |
| **歧义与纠错** | 遇到多同名节点或未选物体直接报错终止 | **跨越轮次的记忆点选**：当你说“让**它**发光”时，系统能结合记忆锁定目标；如果遇到破坏性动作没选中物体，会弹出 **UI 卡片** 向你二次确认。 |
| **操作的边界** | 只能执行预先写好的特定功能面板 | **无限宽广的动态执行流**：遇到没有任何现成工具能做的事（如“帮我把场景里名字带 proxy 的物体全删了并打印清单”），AI 现场写安全 Python 代码并即时运行。 |
| **执行透明度** | 盲盒运行，报错了只能看 Script Editor 的红字 | 每一步操作都伴随**人类可读的结构化回执总结**，并且统统包裹在撤销块（Undo Chunk）中，随时可以后悔。 |

## 📁 目录结构

- `bridge/`：本机 AI 网关（基于 Python 3 + FastAPI），负责对接云端大模型并处理请求。
- `maya_module/`：Maya 插件端核心逻辑代码（包含 UI 面板、沙盒环境与插件业务层）。
- `icon/`：界面所需的图标资源。

## ⚡ 安装与使用全指南

### Step 1: 极速安装 (拖拽即装)
1. 请先确保你的电脑上安装了原生 Python 3 (或 Anaconda 环境)，供本机网关服务使用。
2. 将你下载并解压的插件文件夹中的 `ins_aiformaya.py` 文件，**直接用鼠标拖入 Maya 的主视口 (Viewport) 中**！系统即可全自动跑完安装流程、配置依赖路径并呼出 AI 面板。

### Step 2: 绑定你的 API 密钥
唤出侧边栏聊天面板后：
1. 点击面板右上角齿轮 **⚙️ (设置图标)**。
2. 填入你申请的 **DeepSeek API Key** 或 **Gemini API Key**。
3. 选择对应的大模型版本（推荐日常使用 `deepseek-coder` 或 `deepseek-chat` 体验最佳）。
4. 退回主面板，点击 **[ Start Gateway ]** 按钮，等待几秒钟右上角状态变为绿色的 `Connected`。

### Step 3: 开始向你的专属 TD 提需求!
你可以直接用口语化的中文向它发号施令。
**视频展示常用绝佳案例：**
- 🌟 *“帮我创建一个小球和一个摄像机，并让摄像机始终看向这个球”* （展现上下文+多步骤连贯度）
- 🌟 *“选中这 50 块石头，给我稍微随机打散它们的旋转和大小”* （展现高效工具链+CtrlZ撤销）
- 🌟 *“帮这些零散的方块全部打个组”* （展现无缝打组并自动绝对居中）
- 🌟 *“给这个场景搭一个三点基础布光，再配一个环绕展示摄像机”* （展现 TD 级别的内置资产化）
- 🌟 *“把整个场景都删了”* （触发破坏性防灾拦截，弹出二次确认卡片）

### 🔧 开发者模式 (手动启动网关)
如果你想查看看网关的高级后台报错输出，或需要扩展更多 AI 插件，可以在 PowerShell 独立挂载网关：

```powershell
cd bridge
# 创建并激活虚拟环境
python -m venv .venv
.\.venv\Scripts\activate
# 初始化依赖（首次必须执行）
pip install -r requirements.txt

# 设置密钥后启动 Uvicorn 本机直连服务
$env:DEEPSEEK_API_KEY = "sk-xxxxxxxxxxxxxxxxxxx"
python -m uvicorn server:app --host 127.0.0.1 --port 8765
```
